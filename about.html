Tentang CegahCorona

CegahCorona adalah sebuah projek web yang dapat menampilkan data covid terkini dari API yang tersedia. Adapun tujuan pembuatan web ini adalah untuk memenuhi tugas ujian akhir semester mata kuliah pemrograman web 1 di Universitas Siber Asia sekaligus untuk memperdalam kemampuan dalam membuat web.

Web CegahCorona dibuat menggunakan perpaduan HTML, CSS, dan Javascript. Data corona yang ada di web ini berasal public API yang disediakan oleh covid-19.go.id. JQuery dan graph.js untuk pengolahan data dari API dan menampilkannya pada elemen web. Untuk Layout dan tampilan web diatur menggunakan bootstrap 5.0
